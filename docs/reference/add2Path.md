# Adds a defined variable and value to the global search path

Adds a variable to the global search path of the current environment

## Usage

``` r
add2Path(newPath)
```

## Arguments

- newPath:

  the path that is added

## Author

Chris Reudenbach

## Examples

``` r
if (FALSE) { # \dontrun{
# add path
add2Path('pathtosomewhere')
} # }
```
