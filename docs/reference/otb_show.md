# Print a quick console view of an OTB application's parameters

Print a quick console view of an OTB application's parameters

## Usage

``` r
otb_show(algo, gili = NULL)
```

## Arguments

- algo:

  Character. OTB application name.

- gili:

  Optional list returned by \[linkOTB()\]. If \`NULL\`, \[linkOTB()\] is
  called.

## Value

Invisibly list(caps=..., spec=...)
